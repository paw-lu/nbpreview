A terminal viewer for Jupyter notebooks.

Features
--------

* Render accurate(ish) Jupyter notebooks in your terminal.
  Like cat (or bat) for ipynb files.


Requirements
------------

* Python 3.8+


Installation
------------

You can install *nbpreview* via pip_ from PyPI_:

.. code:: console

   $ pip install nbpreview
